thecodeboss.dev
===

[Aaron Krauss' personal site.](https://thecodeboss.dev)

Built with Wordpress, Docker, Webpack, HTML, Sass, JS, and more.

## To Install
```
nvm use
npm install
cp .env.example .env # And set the variables
```

## To Build
```
npm run dev
# Or npm run watch
# Or npm run prod
```

## To Run
```
docker-compose up
```
